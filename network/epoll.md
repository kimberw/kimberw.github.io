# epoll

## 边缘触发 vs 水平触发

|                    |         边缘触发          |        水平触发        |
| :----------------: | :-----------------------: | :--------------------: |
| 事件触发方式与次数 |  仅在事件到来时触发一次   | 若有事件到达均会被触发 |
|    代码处理 读     | 需要将数据全部读出直至EOF | 会多次触发可以分批读取 |
|      代码难度      |     较复杂，容易写bug     |         较简单         |
